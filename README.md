## Project Overview 
- Create a End-to-End Data Pipeline for Banking Transaction with Real-Time Injestion, Tranformation and Visualization 
- Provide a step  data warehouse and CI/CD 

---

## TechStack 
- **SOURCE LAYER** : ProstgreSQL
- **Streming&CDC** :  Apache Kafka & Debezium
- **Data Lake** : Minio S3 
- **Data Warehouse** :  Snowflake 
- **Tranformation&Processing** : dbt 
- **Orchestration** :  Apache Airflow
- **Visualization tools** : Power Bi
- **CI/CD** :  Docker&Github Action
